== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/sticky-notice-bar` directory, or install via the WordPress Plugins screen.
2. Activate the plugin.
3. Go to **Settings ‚Üí Sticky Notice Bar** to configure your message.

== Frequently Asked Questions ==

= How do I hide the bar after a visitor sees it? =
The bar is automatically hidden once dismissed and only returns when you update the version number.

= Can I show it only on certain pages? =
That feature is available in the Pro version.

= Can I place the bar at the bottom instead of the top? =
Yes, with Sticky Notice Bar Pro.

= Will this work with any theme? =
Yes. The bar is injected using standard WordPress hooks and works with most themes out of the box.

= What are the ‚Äúcoming soon‚Äù Pro features? =
Sticky Notice Bar Pro will soon support visibility on specific pages (e.g., only on homepage or shop) and recurring display logic (e.g., show every Monday).
